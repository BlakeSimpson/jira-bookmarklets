Description

We need to support "like" as verb within our AS implementation.

Only known combinations (object/verb/actor): user/like/jobposting, user/like/article, user/like/companyprofileupdate, user/like/companyprofile, user/like/event, user/like/group, user/like/poll, user/like/thread

Before we can release this feature to production, we should give the mobile team the opportunity to test against all the changes at once. Therefore we have a common branch for all upcoming AS changes: https://source.xing.com/xws/xws/tree/new_as_types
