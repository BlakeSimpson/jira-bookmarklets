Description

We have an old and incorrect statement in our API documentation regarding the test consumer key.

In https://dev.xing.com/docs we write:

Test Consumer Key:
"....All requests to the API will be made via your own XING user ID."

This information is not valid anymore since we removed the restriction to test consumers.

Acceptance criteria:

  • remove the above mentioned statement from the documentation
